# Preprocessing Configuration

## Example:

```
"name": {
    "WTemp" : "Temperature, water"
}
```

- `name` is the observed Property
- `WTemp` is the **Input** Value the FROST Server provides
- `Temperature, water` is the **Output** that is saved in the database

```
"location": {
    "14": [
      9.940,
      54.671
    ],
```

- `location` is the observed Property
- `14` is the **SensorID** Value the FROST Server provides
- `lat, lon` is the **Output** that is saved in the database
